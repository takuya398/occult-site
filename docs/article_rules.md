# Article Rules

## Markdown Only
- Do not use HTML tags in index.md (no span/div/p/h1-h6, no style/class/id).
- Headings must be plain text only.

## Headings
- Use Markdown headings only (e.g., "## 目次", "## 導入").
- Keep heading text simple and consistent.

## Table of Contents
- Use Markdown links only.
- Use anchors without symbols.
  - "アクセス情報・訪問時の注意" -> #アクセス情報訪問時の注意
  - "心霊現象・噂体験談" -> #心霊現象噂体験談

## Styling
- Do not add color or styling in Markdown.
- Heading styling must be done in the renderer (ReactMarkdown components/CSS).
